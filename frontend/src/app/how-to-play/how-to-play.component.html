Enter the Lobby Id

You can either scan the QR Code on the screen or input the code by yourself. Ask the VR Player to give you a lobby ID if you can't see one.

Enter a name

Choose a name that you want to go by. Best choose a name that the VR Player knows you as since they can't hear you during the game.

Let the fun begin

Start chatting with the other players and read through the diary to find extra clues for the VR Player.

VR

Setup

Start the application and wear the setup as described by the producer. This game was developed for Oculus Quest 2. Start the game and wait a few seconds for the other players, so that they can enter the Lobby ID on the website.
